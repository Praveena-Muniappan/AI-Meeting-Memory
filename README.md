# AI Meeting Memory

An AI-powered meeting assistant that automatically transcribes meetings, generates concise summaries, extracts action items, and enables semantic search across previous discussions.

---

## Features

- 🎙️ Upload meeting audio files
- 📝 Automatic speech-to-text transcription using Whisper
- 🤖 AI-generated meeting summaries
- ✅ Action item extraction
- 📌 Key decision identification
- 🔍 Semantic search across previous meetings
- 💾 SQLite database for meeting history
- 📄 PDF export for meeting reports

---

## Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **Speech Recognition:** Whisper
- **LLM:** Google Gemini API
- **Database:** SQLite
- **Embeddings:** Sentence Transformers
- **Vector Search:** FAISS
